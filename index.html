About

What is DriveRent?

DriveRent is a command-line Java application designed to efficiently manage a car rental business. It's fast, lightweight, and runs directly in the terminal.

Designed for simplicity

The application allows easy input, real-time status updates, and summary reports on rentals, vehicles, and customers. All data is stored locally in CSV files for transparency and easy maintenance.

Modern features

DriveRent includes advanced features such as sorting, searching, and filtering to help users quickly find the data they need and streamline rental operations.

Why us

Keep everything in order — from available vehicles and customer reservations to rental history and statistics — all in one place.

Statistics

Get a clear view of rental trends, most popular cars, and performance over time through automatically generated summaries.

User interface
